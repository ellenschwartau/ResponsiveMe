Module: allMediaQueries

Dieses Modul dient der Visualisierung von Media Queries. Es liest die in den Style Sheets der aufgerufenen Website enthaltenen Media Queries aus und bringt diese zur Anzeige. Über einen Mausklick auf den Selektor der Media Query kann anschließend an die Stelle gesprungen werden, an der diese Media Query greift. Die Media Queries können außerdem über die vorhandenen Code Editoren bearbeitet werden.
